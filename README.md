## 文档目录规范
1. 所有的导言内容均统一写在 `premble.tex` 中
1. 讲义整理的内容放在 `Lecture/` 中, 文档命名方法为 `page_<pagenum>`, 其中 `<pagenum>` 指对应的文件页码.

## LaTeX 指令规范
主要是关于 `premble.tex` 里定义的几条自定义指令的使用规范.
1. 作为圆周率的 $\pi$ 写为 `\MaPI`
1. 作为自然常数的 $e$ 写为 `\MaE`
1. 表示微小增量的符号写为 `\Dbar`

## LaTeX 文件编写规范
1. 作为调试中使用的主文件的名称为 `main.tex`. 由于已将该文件名加入了 `.gitignore`, 故不必担心会被 git 检测到更新.
1. 仅上传 `Lecture/` 内的文件.
